# Luna MVP Demo v4

A fresh iteration of the Luna interaction demo, building on the Facebook SwiftUI Template for rapid iOS prototyping.

## Project Overview
This is version 4 of the Luna MVP demo, created as a clean slate to continue development while preserving the v3 work.

## **Instructions**
- **Download Xcode** from **Managed Software Center** (Xcode 26.0.0_17A324_fb.app)
- After downloading, **open Xcode** and **install the iOS 26 simulator**
- After installing, **open luna-mvp-demo-v4.xcodeproj**
- Select your preferred simulator / device on the top center, and **click ▶️ to run the app**

---

## **Vibe coding in Cursor**
- **Request to join [Cursor Users](https://fb.workplace.com/groups/725594866994618)** to get access to Cursor
- After being accepted, **download** and **install Cursor** by signing up with your @meta email
- **Open the luna-mvp-demo-v4 folder** in Cursor
- **Prompt Cursor** to make changes
- **Return to Xcode** and **click ▶️ to re-build the app**
### Optional
- To build and run without opening Xcode, **edit the .vscode/tasks.json** and **swap in the paths** to your project and simulator of choice
- Then **press COMMAND + SHIFT + P** and select "**Tasks: Run Build Task**" after making your changes

## **Vibe coding in VS Code @ FB**
- **Open the luna-mvp-demo-v4 folder** in **VS Code @ FB**
- Select **Devmate** in the left navigation bar
- **Prompt Devmate** to make changes
- **Return to Xcode** and **click ▶️ to re-build the app**
### Optional
- To build and run without opening Xcode, **edit the .vscode/tasks.json** and **swap in the paths** to your project and simulator of choice
- Then **press COMMAND + SHIFT + P** and select "**Tasks: Run Build Task**" after making your changes
